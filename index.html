No JS multiple color scheme picker

This color scheme toggle uses just html and css to change the color of the page.

It also uses JavaScript to add progressive enhancements — like storing user's color scheme preferences, and updating the theme-color meta tag.

This was created for a tutorial you can read here or on dev.to

You can see a real life of this in action at missingdice.com

Much of this tutorial is an extension of a previous tutorial about an html and css only dark mode toggle — also available on dev.to.
